How to run (Node.js)

1. Open a terminal and go to the backend folder:
   cd ai-tools-website/backend

2. Install dependencies:
   npm install

3. Start the server:
   npm start

4. Open the site in your browser:
   http://localhost:3000

Notes:
- The backend stores data in memory (demo). Restarting the server resets the tools list.
- You can edit server.js to add persistent storage (file or database).
- To stop the server press CTRL+C in your terminal.
